# 🛠️ Assessment Project - Local Setup Guide

## 📦 Prerequisites

Ensure the following are installed on your system:

- [Node.js](https://nodejs.org/) v22
- [npm](https://www.npmjs.com/) (Node.js package manager)
- [Git](https://git-scm.com/)

## 🚀 Quick Start

### 1. Clone the Repository

```bash
git clone https://bitbucket.org/aptoslabs05/aptos-assessment.git

cd aptos-assessment
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the project

```bash
npm start
```
